Credits

Pusaka Framework di ciptakan pertama kali oleh seorang developer bernama Tom Richard. Pusaka Framework di tulis dan dikembangkan berdasarkan kebutuhan di dunia nyata khususnya dunia pekerjaan yang ada di indonesia. Dengan mengamati dan mempelajari kebutuhan tersebut maka diciptakanlah Pusaka Framework dengan sejumlah library dan extension serta tools Command Line yang dibuat original sehingga dapat memudahkan developer untuk membuat sebuah project.

Awal pembuatan Pusaka Framework bermula pada tahun 2016 dengan versi 1.0.

Pada tahun 2020 Pusaka Framework release dengan versi 3.0 Beta

Sampai sekarang Pusaka Framework masih dikembangkan oleh satu orang dan belum memiliki kontributor, semoga kedepan bisa mendapatkan kontributor.
