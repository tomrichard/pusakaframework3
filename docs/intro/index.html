Pusaka Framework 3

Pusaka Framework adalah sebuah Application Development Framework atau sebuah alat (toolkit) yang digunakan untuk kepentingan membuat sebuah Website menggunakan PHP. Tujuan akhir dari framework ini adalah mempermudah dan mempercepat pengerjaan sebuah website dengan banyaknya tools dan library yang disediakan sesuai kebutuhan, sehingga penulisan kode tidak lagi dimulai dari kosong (scratch). Pusaka Framework membuat anda fokus kepada project anda dengan meminimalisir jumlah penulisan kode.

Pusaka Framework juga bersifat flexibel dimana anda dapat menambahkan (extends) beberapa part dari core system sesuai kebutuhan yang anda inginkan.

Mengapa Menggunakan Pusaka Framework ?

Pusaka Framework cocok untuk anda jika :

  • Baru mulai membuat website.
  • Menginginkan sebuah Framework yang mempunyai size yang kecil (small footprint).
  • Menginginkan performa yang baik (load speed).
  • Menginginkan tingkat konfigurasi yang rendah (zero configuration).
  • Tidak membutuhkan Command Line.
  • Tidak ingin dipaksa mengikuti aturan templating engine.
  • Membutuhkan solusi code untuk tingkat kompleksitas tinggi.
  • Membutuhkan dokumentasi lengkap.
